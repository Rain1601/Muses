# 工具栏交互测试场景

## 基础功能测试
- [ ] 鼠标拖拽选择文本 → 工具栏出现
- [ ] 键盘 Shift+方向键选择 → 工具栏出现
- [ ] 选择少于3个字符 → 工具栏不出现
- [ ] 点击工具栏外部 → 工具栏关闭
- [ ] 点击工具栏内部 → 工具栏保持

## 输入测试
- [ ] 输入字母数字 → 显示在工具栏输入框
- [ ] 输入中文 → 正确处理
- [ ] Backspace → 删除输入框字符，不删除选中文本
- [ ] Delete → 被拦截，不删除选中文本

## 快捷键测试
- [ ] Cmd+B → 文本加粗
- [ ] Cmd+I → 文本斜体
- [ ] Cmd+U → 文本下划线
- [ ] Cmd+Shift+H → 文本高亮
- [ ] Cmd+C → 复制选中文本
- [ ] Cmd+V → 粘贴（工具栏关闭后）

## 工具栏操作
- [ ] 选择动作 → TextActionDialog 出现
- [ ] 复制按钮 → 文本复制到剪贴板
- [ ] Enter → 执行选中的动作
- [ ] Esc → 关闭工具栏
- [ ] 方向键 → 导航选项

## 边界场景
- [ ] 二次选择相同文本 → 不闪烁
- [ ] 快速切换选择 → 工具栏平滑过渡
- [ ] 屏幕边缘选择 → 工具栏位置自动调整
- [ ] 长文本选择 → 工具栏预览截断
- [ ] AI 禁用状态 → 显示禁用提示

## 性能测试
- [ ] 大文档选择 → 响应速度正常
- [ ] 频繁选择/取消 → 无内存泄漏
- [ ] 多个编辑器实例 → 互不干扰
